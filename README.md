Port Scanner

Um Port Scanner simples desenvolvido em Java para identificar portas abertas em um host específico.
Este projeto foi desenvolvido com fins educacionais, para estudar conceitos de redes e programação em Java.

📌 Descrição

O port scanner realiza tentativas de conexão em diferentes portas de um host.
Quando a conexão é estabelecida com sucesso, a porta é considerada aberta.

Ferramentas desse tipo são utilizadas em:

Testes de segurança

Administração de redes

Diagnóstico de serviços em servidores

Aprendizado de programação de redes

⚙️ Tecnologias utilizadas

Java

JDK 21

Classe Socket da biblioteca padrão Java
